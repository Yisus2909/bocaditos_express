# 🥟 Bocaditos Express - Landing Page (AI & Tailwind CSS)

Un template ultraligero, moderno y 100% responsive creado específicamente para negocios locales de comida. 

Este proyecto fue estructurado utilizando **Inteligencia Artificial** como herramienta de pair-programming para acelerar el maquetado, y está estilizado con **Tailwind CSS** vía CDN para garantizar una carga instantánea sin necesidad de configuraciones complejas de entorno.

## 🚀 Características Principales

* **⚡ Carga Ultrarrápida:** Cero dependencias pesadas, cero frameworks innecesarios. Solo HTML puro, Tailwind CSS y Vanilla JS.
* **🌙 Dark/Light Mode:** Integración de modo oscuro nativo que respeta las preferencias del sistema del usuario.
* **📱 Mobile-First:** Diseño fluido que se adapta perfectamente a celulares y pantallas de escritorio.
* **💬 Botón de Conversión:** Integración de un Floating Action Button (FAB) directo a WhatsApp.
* **🎥 Hero Dinámico:** Soporte nativo para videos en bucle en la sección principal.
* **🎨 Diseño Artesanal:** Fuentes serifadas y paletas de colores cálidos para evocar comida casera.
